# Media Plan

Dashboard for product cards and AB-test analytics.

## Local Run

```bash
npm install
npm run dev
```

Open:

- `http://127.0.0.1:5173/`
- `http://127.0.0.1:5173/ab-tests`
- `http://127.0.0.1:5173/ab-tests-xway`
- `http://127.0.0.1:5173/cards`

## Main Areas

- product cards
- AB-test dashboard from table export
- AB-test dashboard from XWAY
- Cloudflare Pages Functions for auth, state, WB, and XWAY API access
